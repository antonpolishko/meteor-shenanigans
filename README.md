# meteor-shenanigans

### Pre-requsites

Install meteor
``` 
make install-meteor
```

Clone the todo-app example and run it
```
git clone https://github.com/meteor/simple-todos-react.git
cd simple-todos-react/
meteor npm i
meteor run
```

Create an electron desktop app:
```
make create-app
```

Open app
```
make open-app
```
